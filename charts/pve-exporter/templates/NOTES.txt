Then open:
http://localhost:{{ .Values.service.port }}/pve?target={{ .Values.proxmox.host }}

Prometheus integration:
Check Prometheus UI -> Targets

Look for:
- job: pve-exporter
- state: UP

Test query:
pve_up

Expected result:
- 1 -> exporter working
- 0 -> exporter reachable but Proxmox failing

--------------------------------------------------

Troubleshooting:
1. Target not visible in Prometheus
   Check ServiceMonitor label:
   release: kube-prometheus-stack
2. Target DOWN
   Verify Proxmox IP: {{ .Values.proxmox.host }}
   Check firewall and API access
3. Auth issues
   Inspect logs:
   kubectl logs -n {{ .Release.Namespace }} deploy/{{ include "pve-exporter.fullname" . }}
4. Secret not created
   Ensure Sealed Secrets controller is running
